>[Vespa](https://vespa.ai/) is a fully featured search engine and vector database.
> It supports vector search (ANN), lexical search, and search in structured data, all in the same query.

## Installation and Setup


<CodeGroup>
```bash pip
pip install pyvespa
```

```bash uv
uv add pyvespa
```
</CodeGroup>
