>[spaCy](https://spacy.io/) is an open-source software library for advanced natural language processing, written in the programming languages Python and Cython.

## Installation and Setup


<CodeGroup>
```bash pip
pip install spacy
```

```bash uv
uv add spacy
```
</CodeGroup>
