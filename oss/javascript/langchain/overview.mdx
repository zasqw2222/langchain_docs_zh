LangChain 是开始构建由大型语言模型（LLM）驱动的代理和应用程序的最简单方式。只需不到 10 行代码，您就可以连接到 OpenAI、Anthropic、Google 等平台。 [more](/oss/javascript/integrations/providers/overview).LangChain 提供了预先构建的代理架构和模型集成，帮助您快速入门，并无缝地将大语言模型（LLMs）集成到您的代理和应用程序中。


我们建议您使用 LangChain 来快速构建代理和自主应用程序。[LangGraph](/oss/javascript/langgraph/overview),我们的低级代理编排框架和运行时，适用于需要结合确定性和智能代理工作流、高度定制以及严格控制延迟的更高级需求。

LangChain [代理](/oss/javascript/langchain/agents) 构建在 LangGraph 之上，以提供持久化执行、流式处理、人工介入、持久化等功能。对于基本的 LangChain 代理使用，您无需了解 LangGraph。

## <Icon icon="wand-magic-sparkles" /> 创建代理



```ts
import * as z from "zod";
// npm install @langchain/anthropic to call the model
import { createAgent, tool } from "langchain";

const getWeather = tool(
  ({ city }) => `It's always sunny in ${city}!`,
  {
    name: "get_weather",
    description: "Get the weather for a given city",
    schema: z.object({
      city: z.string(),
    }),
  },
);

const agent = createAgent({
  model: "claude-sonnet-4-5-20250929",
  tools: [getWeather],
});

console.log(
  await agent.invoke({
    messages: [{ role: "user", content: "What's the weather in Tokyo?" }],
  })
);
```
